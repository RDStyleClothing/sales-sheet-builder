Folder of garment images. Use STYLE - Color Name - Color Code - Fabric Content.ext to autofill every field per style — e.g. 78MS301S - BLACK - H34 - 100 COTTON.png. Add a fifth segment for the fabric description (e.g. ... - 50 COTTON 50 ACRYLIC - GARMENT WASH.png). The shorter STYLE - Color Name - Color Code.ext still works, as does the dash-suffix variant 58MT400S - SAGE-B-2182.png and legacy STYLE_COLOR.ext.
